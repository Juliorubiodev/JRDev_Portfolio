# Julio Rubio - Computer Engineer Portfolio

Welcome to the source code of my personal portfolio. This project is a showcase of my work, skills, and experience as a Computer Engineer and Full Stack Developer.

## 🚀 About The Project

This portfolio is a modern, responsive web application designed to highlight my professional journey, featured projects, and technical expertise. It features a clean, dark-themed UI with glassmorphism effects, smooth animations, and full internationalization support (English/Spanish).

### Key Features
- **Modern UI/UX**: Built with a focus on aesthetics and user experience using Tailwind CSS and Framer Motion.
- **Internationalization (i18n)**: Fully bilingual support (EN/ES) with seamless language switching.
- **Responsive Design**: Optimized for all devices, from mobile phones to large desktop screens.
- **Dynamic Content**: Project cards, experience timelines, and tech stack sections are data-driven.
- **Contact Integration**: Easy-to-use contact buttons and email copy functionality.

## 🛠️ Built With

This project relies on a robust stack of modern web technologies:

-   **[React](https://react.dev/)**: Frontend library for building the UI.
-   **[TypeScript](https://www.typescriptlang.org/)**: For type-safe code and better developer experience.
-   **[Vite](https://vitejs.dev/)**: Next-generation frontend tooling for fast builds.
-   **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.
-   **[Shadcn UI](https://ui.shadcn.com/)**: Reusable components built with Radix UI and Tailwind CSS.
-   **[Framer Motion](https://www.framer.com/motion/)**: For production-ready animations.

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v18+ recommended)
*   npm or yarn

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Juliorubiodev/portfolio.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```
